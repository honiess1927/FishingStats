This is a requirement doc that describes the requirement for adding a new functionality to the FishingStatus plugin. This new functionality enables the user to view the estimated hourly earn by region.



1. Add a button to the right side of the panel. You can give it a proper name.
2. On clicking the button. It opens up a new popup window. 
3. The popup window display an overview page of estimated hourly earn fishing in each region.
4. There's anoter tab for "details" which provides a dropdown that let the user view info by region. The detail view shows the total number of fish captured for this region by type, the percentage, and the estimated earn. (Calculated by 300 fish catch per hour)
